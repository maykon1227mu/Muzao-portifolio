Especialidades

Um pouco sobre minhas habilidades


JavaScript é uma linguagem de programação usada para criar interatividade e funcionalidades dinâmicas em sites. Inicialmente desenvolvida para o lado do cliente, agora também é usada no lado do servidor com Node.js.

HTML é a linguagem de marcação usada para estruturar o conteúdo da web. Define a organização de elementos como títulos, parágrafos, links e imagens, formando a base de qualquer página web.

CSS é a linguagem de estilo usada para descrever a aparência das páginas web. Permite definir cores, fontes, layouts e outras características visuais, separando a apresentação do conteúdo.

PHP é uma linguagem de script do lado do servidor, usada para gerar conteúdo dinâmico em páginas web. Facilita a interação com bancos de dados e a gestão de sessões de usuário, sendo amplamente utilizada no desenvolvimento web.

React é uma biblioteca JavaScript desenvolvida pelo Facebook para criar interfaces de usuário dinâmicas e interativas. Utiliza componentes reutilizáveis e um "virtual DOM" para melhorar a eficiência e desempenho das atualizações na interface.
